# realtime_vad_sample
A simple sample code of real-time voice activity detection using webrtcvad.

# Environment
This worked on Windows 10 and Ubuntu 18.04.
I think this will work on MacOS.

# Installation
## numpy
```bash
pip install numpy
```
### SoundCard
```bash
pip install SoundCard
```
### webrtcvad
```bash
pip install webrtcvad
```
# Usage
```bash
python vad.py
```
