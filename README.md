# realtime-vad-sample
Sample code of real-time voice activity detection using webrtcvad.

# Environments
This worked on Windows 10/11 and Ubuntu 18.04.
I think this will work on macOS.

# Installation
## numpy
```bash
pip install numpy
```
### SoundCard
```bash
pip install SoundCard
```
### webrtcvad
```bash
pip install webrtcvad
```
# Usage
```bash
python vad.py
```
